# PlayCafé

**A responsive website interface with the use of front-end framework (Bootstrap), without back-end scripting.**

- Search filters
- Responsive webpage
- Customer reviews
- Social media
- Google Map Location
- Photos and Images
- Calendars - Reservation
- Enquiry Form
